A website is a collection of web pages that are connected and can be accessed through the internet using a web browser.

Key Points:

Made of web pages (HTML, CSS, JavaScript)
Can include text, images, videos, forms, and links
Has a domain name (like example.com)
Stored on a web server
Accessible online 24/7

Types of Websites:

Static Website – fixed content, rarely changes
Dynamic Website – content changes based on user interaction
E-commerce Website – online stores
Blog/Portfolio – personal or professional content
Corporate/Business Website – for companies and organizations
