A web server is a system that receives requests from web browsers and sends back web content (like web pages, images, or data) over the internet.

explanation:

A web server is like a delivery service for websites.
Your browser asks for a page, and the web server delivers it.

What a web server can do:
Serve HTML, CSS, JavaScript files
Deliver images, videos, and documents
Handle user requests (logins, forms)
Work with programming languages like PHP, Python, Java, Node.js

Types of web servers:

Apache
Nginx
Microsoft IIS
LiteSpeed
