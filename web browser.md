A web browser is a software application that lets you access, view, and interact with websites on the internet.

explanation:

A web browser is the tool you use to open and use the web.

What a web browser does:

You type a URL (website address)
The browser sends a request to a web server
The browser receives the response
It displays the web page for you to see and use

Common web browsers:

Google Chrome
Mozilla Firefox
Microsoft Edge
Safari
Opera
