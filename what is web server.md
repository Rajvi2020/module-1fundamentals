A web server is a system that stores, processes, and delivers web pages to users over the internet.
When you open a website in your browser, youâ€™re interacting with a web server.

How a web server works:
You type a website address (URL) in your browser
Your browser sends a request to a web server
The web server finds or generates the requested content
The server sends the content back to your browser
Your browser displays the web page

What a web server can serve

Web pages (HTML, CSS, JavaScript)
Images and videos
Data (JSON, APIs)
Dynamic content (generated using PHP, Python, Node.js, etc.)
